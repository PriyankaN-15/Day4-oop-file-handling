 Day 4 - Student Grade Manager

 Overview
This is a Python-based mini project that manages student records using Object-Oriented Programming, file handling, and error handling.

 Features
- Add student with grades
- Calculate average marks
- View all student records
- Save and load data using JSON
- Handles invalid input gracefully

 Concepts Used
- OOP (Classes, __init__, methods)
- File Handling (read/write JSON)
- Error Handling (try/except)

 How to Run
1. Run the program:
   python manager.py

2. Follow menu options

 Learning Outcome
- Built a real-world Python project
- Combined multiple concepts into one application
- Improved problem-solving and code structure
